Python 3.6.15 For Windows XP SP3

WinXP_Python-3.6.15 is the latest version of Python 3.6.x (3.6.15 as of 12/24/2021) and is compatible with Windows XP SP3 32-bit systems. The last official Python version for Windows XP is Python 3.4, so by using WinXP_Python-3.6 you have access to many more Python applications then ever before on Windows XP SP3.

Building

WinXP_Python-3.6.15 requires the last compatible Cygwin for Windows XP SP3 to be installed (32-bit) (64-bit).

Start Cygwin Setup

Run the setup exe from cmd.exe with the -X flag:

32-bit: 

setup-x86-2.874.exe -X

64-bit: 

setup-x86_64-2.874.exe -X

Add The Windows XP SP3 Compatible Repo

32-bit: 

http://ctm.crouchingtigerhiddenfruitbat.org/pub/cygwin/circa/2016/08/30/104223

64-bit: 

http://ctm.crouchingtigerhiddenfruitbat.org/pub/cygwin/circa/64bit/2016/08/30/104235

Install Dependencies & Compile

Install all of the devel packages using the Cygwin Setup. Then start the Cygwin Terminal. Cd into the winxp_python-3.6.15-1 directory, and ./configure && make install like you do building any other Python release.

After doing a make install, you need to 'rebaseall' to fix some ugly warnings when using Python3.6.exe. To do so:

  • In the Cygwin Terminal, do: 
    find /usr/local -type f -name "*.dll" > /tmp/rebase-list
  • Close the Cygwin Terminal and all other Cygwin programs.
  • Open cmd.exe and do: 
    cd C:\cygwin\bin && C:\cygwin\bin\ash.exe C:\cygwin\bin\rebaseall -T C:\cygwin\tmp\rebase-list && del C:\cygwin\tmp\rebase-list
